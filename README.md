# "how-to-learn"

"how-to-learn" это учебный проект в виде сайта от Яндекс Практикума созданный мной Марокко Тимуром под внимательным руководством наставников Практикума, для ознакомления современными технологиями верстки.

## Installation and Usage

Для того, чтобы открыть сайт у себя на компьютере или другом устройстве, просто скачайте себе директорию по данной [ссылке](https://github.com/proofblame/how-to-learn.git) и откройте файл Index.html в любом браузере.

```bash
Index.html
```

## Technologyes

В проекте применены следующие технологии:
1. HTML5;
2. CSS3;
3. Animations;
4. BEM technology;
5. File structure - NESTED.
